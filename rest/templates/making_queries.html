Basic Query Methods

All()

Returns every object from the database table. Equivalent to a SQL SELECT * query.

See more

Get()

Fetches exactly one object matching the condition. Raises an error if none or multiple are found.

See more

Filter()

Returns a QuerySet of objects that match the specified lookup conditions.

See more

Exclude()

Filters out records that meet the given criteria, returning all others.

See more

Order By()

Orders the returned records by one or more fields, ascending or descending.

See more

Slice[]

Restricts the QuerySet to a certain number of results, similar to SQL LIMIT.

See more

Exists()

Returns True if the QuerySet contains any results, without fetching them.

See more

Advanced Query Methods

Select Related()

Performs a SQL JOIN to fetch related objects in one query, reducing database hits.

See more

Prefetch Related()

Fetches related objects for ManyToMany or reverse relations using separate optimized queries.

Q()
